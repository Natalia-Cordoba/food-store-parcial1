# 🍕 Food Store - Parcial 1 Programación 3

Proyecto realizado para la Evaluación 1 de la materia Programación 3.

## 📌 Descripción

Food Store es una aplicación frontend desarrollada con HTML, CSS, JavaScript y TypeScript que simula una tienda de comidas online.

El proyecto permite:

- Visualizar productos dinámicamente
- Buscar productos por nombre
- Filtrar productos por categoría
- Agregar productos al carrito
- Persistir el carrito utilizando localStorage
- Visualizar el detalle del carrito
- Modificar cantidades de productos
- Eliminar productos del carrito
- Vaciar el carrito
- Calcular el total de la compra

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- TypeScript
- Vite
- localStorage

---

## ▶️ Instrucciones para ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/Natalia-Cordoba/food-store-parcial1.git
```

2. Instalar dependencias:

```bash
pnpm install
```

3. Ejecutar el proyecto:

```bash
pnpm dev
```

4. Abrir en el navegador:

```txt
http://localhost:5173
```
